# Krug-Informed UX Gate

These are Krug-derived usability heuristics plus Fredo-specific premium requirements.

## Core usability

- [ ] First-time visitor understands what this site is within seconds.
- [ ] Pages are self-evident.
- [ ] Visual hierarchy communicates first, second and third priority.
- [ ] Clickable/tappable things look actionable.
- [ ] Familiar conventions are used where they help.
- [ ] Pages are scannable without reading everything.
- [ ] The primary next action is obvious.
- [ ] Forms ask only what is needed at that stage.
- [ ] Page/section names match what users selected.
- [ ] Logo/home behavior is predictable.
- [ ] Current location/context is clear.
- [ ] Users always have an escape route.
- [ ] The homepage explains the site's purpose and main choices.
- [ ] The primary task is never buried behind cinematic effects.
- [ ] Trunk test passes: identity, location, sections, navigation, next action.

## Fredo-specific clarity

- [ ] Art appears before sales pressure.
- [ ] Available, sold/private, print-only and licensing states are unmistakable.
- [ ] Make-an-offer never implies automatic acceptance.
- [ ] Commission/mural requests state custom quote only.
- [ ] Licensing requests require human negotiation.
- [ ] Chilean Spanish is natural and primary.
- [ ] English preserves meaning rather than translating mechanically.

## Premium additions

- [ ] Motion helps meaning and never blocks content.
- [ ] Reduced-motion mode preserves the full experience.
- [ ] Gallery remains fast and usable without animation.
- [ ] Mobile is first-class, not a cropped desktop layout.
- [ ] No fake metrics, fake press, fake scarcity, fake buyers or fake reviews.
- [ ] Typography, spacing and imagery feel editorial/museum-grade.
- [ ] No generic glassmorphism, random gradients, floating particles, SaaS card grids or gratuitous 3D flips.
